# 📱 PersonalBanner — Android Install Guide (No PC Needed)

You only need **2 files**: `manifest.json` and `index.js`

---

## Step 1 — Host the files on GitHub (free)

1. Go to **github.com** on your phone browser
2. Sign in or create a free account
3. Tap **+** → **New repository**
   - Name it: `PersonalBanner`
   - Set it to **Public**
   - Check ✅ "Add a README file"
   - Tap **Create repository**

4. Inside the repo, tap **Add file → Upload files**
5. Upload both `manifest.json` and `index.js`
6. Tap **Commit changes**

7. Now go to **Settings → Pages** in the repo
   - Under "Branch" select `main` → `/ (root)` → **Save**
   - GitHub will give you a URL like:
     `https://yourusername.github.io/PersonalBanner`

---

## Step 2 — Install in Revenge

1. Open Discord with Revenge installed
2. Go to **Settings → Revenge → Plugins**
3. Tap the **+** or **Install Plugin** button
4. Paste your GitHub Pages URL:
   ```
   https://yourusername.github.io/PersonalBanner
   ```
5. Tap **Install** → then **Enable** ✅

---

## Step 3 — Set your banner

1. Next to the plugin tap the ⚙️ gear icon
2. Paste a direct image/GIF URL into the banner field
3. Pick your height (80 / 120 / 160 / 200 px)
4. Toggle rounded corners on/off
5. Open your profile to see it live 🎉

---

## 🔗 Where to get GIF/Image URLs

| Site | How |
|------|-----|
| **Tenor** | Open a GIF → Share → Copy Link (grab the `.gif` URL) |
| **Giphy** | Open a GIF → Share → Copy GIF Link |
| **Imgur** | Upload an image → right-tap → copy image address |
| **Discord** | Upload to any channel → long-press → Copy Media Link |

> ✅ The URL must end in `.gif`, `.png`, `.jpg`, or `.webp`
> ❌ Don't use a webpage link — it must be a direct media link

---

## ❓ Troubleshooting

| Problem | Fix |
|---------|-----|
| Banner doesn't show | Make sure you opened **your own** profile, not someone else's |
| Image doesn't load | Check the URL is a direct media link (ends in .gif/.png etc.) |
| Plugin won't install | Make sure GitHub Pages is enabled and the URL has no trailing slash |
